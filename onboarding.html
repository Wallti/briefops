02
Calibration

Pipeline calibration on your sample meetings.

We run your three sample meetings through the full pipeline. You review the extracted tasks and flag any routing errors. The parser is adjusted to your team's language and terminology.

03
Staging

End-to-end test in staging environment.

Full pipeline test: meeting input → task extraction → Jira staging board → Slack staging channel → calendar test event. You verify the output looks correct in each target system.

Standard + Advanced · Additional steps

What changes
for higher tiers.

Standard · Days 5–14
Dependency graph configuration — cross-team blocking rules based on your org-chart
Multi-channel delivery tuning — Slack channels mapped per team, email templates reviewed
Approval workflow configuration — who approves what, escalation thresholds
Blocker detection calibration — test cases for your most common cross-team dependencies
Advanced · Days 11–25
Knowledge base initial ingestion — document library processed and indexed
Document privacy classification — each document assigned full / reference-only / invisible
Capacity analysis calibration — sprint history loaded, estimation model trained
Executive digest configuration — recipients, format, delivery schedule
Follow-up agent calibration — escalation thresholds tuned to your team's rhythm
